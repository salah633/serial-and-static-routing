# serial-and-static-routing
## Description:-
- it is a network of 4 routers three in the same line and the fourth is connected to the middle one
- using ip like 10.0.0.0 in each network.
- the middle router is connected to first router with cross over and the others with serial cable.
- each router has his network(switch and device)
- so performing static and serial routing is tested.

  ## steps to run:-
   - you can download this file.
   - you can click on it to run or open cisco packet tracer to run it.

  ## testing  Connectivity:-
   - Open the `Command Prompt` on any PC within the network.
   - Use the `ping` command to test connectivity to any other device (40.0.0.2).
